- Clone the project

```bash
  git clone https://github.com/coen-h/zmov
```

- Go to the project directory

```bash
  cd zmov
```

- Install dependencies

```bash
  npm install
```

- Create a .env file

- Request an API key from TMDB and them add it to the .env file

```
VITE_API_KEY=VITE_API_KEY
```

- Start the server

```bash
  npm start
```
